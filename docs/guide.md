## 📦 安装

```bash
npm install dessert-ui
```

```bash
yarn add dessert-ui
```

```bash
pnpm add dessert-ui
```

## 🔨 使用

```tsx
import { Add } from 'dessert-ui/Icon';

export default () => (
  <>
    <Add />
  </>
);
```
